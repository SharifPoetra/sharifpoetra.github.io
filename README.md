## 🛠️ Built With (Modern Tech Stack)

The entire architecture has been migrated to a fully typed, performance-optimized, and component-driven structure:

- **Framework:** [Next.js](https://nextjs.org/) (v15+ App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/) (Strictly Typed)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) (Using CSS-first `@import "tailwindcss"` engine)
- **Deployment:** [GitHub Pages](https://pages.github.com/) via automated **GitHub Actions** workflows

## 🚀 Local Development

To run this Next.js app locally and experiment with the codebase:

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/SharifPoetra/sharifpoetra.github.io.git](https://github.com/SharifPoetra/sharifpoetra.github.io.git)
   cd sharifpoetra.github.io
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

   Open http://localhost:3000 inside your mobile or desktop browser to see the results.
## 📦 Automated Deployment
This repository uses a custom CI/CD pipeline tailored for modern static sites. Every time code is pushed to the master branch, a GitHub Action automatically triggers to:
 1. Compile and audit TypeScript schemas.
 2. Build and export static structures via Next.js output: 'export' into raw HTML assets.
 3. Automatically deploy production bundles directly onto GitHub Pages.
